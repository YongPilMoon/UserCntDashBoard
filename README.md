# User Count DashBoard
시간별 사용자 유입수를 보여주는 dashboard 입니다. 표와 그래프 형식으로 보여주며 백엔드에는 node.js 프론트에는 react.js를 사용하였습니다.

## Core Technology
이 프로젝트에선 다음과 같은 기술들이 사용되었습니다.

### Backend
- node.js
- express

### Frontend
- react.js
- redux
- immutable.js
- axios
- redux-promise-middleware
- Sass
- webpack
- create-react-app
- babel
- chart.js
